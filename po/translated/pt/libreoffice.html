LibreOffice (Backports)

Quando você clica no botão instalar todos os aplicativos listados serão instalados no seu sistema. Você pode usar o gestor de software para instalá-los separadamente.
LibreOffice

SolydXK vem com a suíte LibreOffice pré-instalada. A sua estabilidade e características funciona muito bem para a maioria das pessoas e especialmente, para as empresas. No entanto, se você está faltando alguns recursos ou você quer o LibreOffice para se manter atualizado você pode instalar o LibreOffice a partir do repositório backports.

Backports são novos pacotes recompilados para o ambiente estável atual. Então, eles vão correr sem novas bibliotecas.

Embora você possa executar mais atualizações de software usando o repositório backports, você introduzir um maior risco de quebra ao fazê-lo.
