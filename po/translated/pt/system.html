Aplicações do sistema

Quando você clica no botão instalar todos os aplicativos listados serão instalados no seu sistema. Você pode usar o gestor de software para instalá-los separadamente.
Gparted

GParted é um editor de partições livre para gerir graficamente as partições do disco. Com GParted você pode redimensionar, copiar e mover partições sem perda de dados.
Página inicial: http://gparted.sourceforge.net
VirtualBox

VirtualBox é um virtualizer completo de propósito geral para hardware x86, voltado para servidor, desktop e uso embutido. O pacote VirtualBox instala num sistema operacional de host existente como um aplicativo; este aplicativo host permite que os sistemas operacionais convidados adicionais, cada um conhecido como um sistema operacional do cliente, para carregar e executar, cada um com seu próprio ambiente virtual.
Página inicial: https://www.virtualbox.org
FileZilla

FileZilla FTP Client é um cross-plataforma rápida e confiável, FTPS e SFTP cliente com bastantes características úteis e uma interface de utilizador gráfica intuitiva.
Página inicial: https://filezilla-project.org
