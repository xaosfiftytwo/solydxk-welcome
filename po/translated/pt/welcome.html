Bem-vindo ao SolydXK

Obrigado por usar SolydXK

SolydXK é um sistema operativo de código aberto de ambiente de trabalho para empresas, organizações e utilizadores domésticos.

Por favor, considere uma doação. Sua doação é muito apreciada e assegura que possamos continuar a melhorar SolydXK.

Nós criamos uma lista de software recomendado para empresas e utilizadores domésticos que você pode instalar nas páginas seguintes.

Clique no butão Seguinte para começar.

Software

Todo o software é construído, mantido e controlado pela comunidade. Base de dados de software chamados "repositórios" são usados para oferecer e manter mais de 60.000 pacotes de software. Assim, não há necessidade de descarregar software a partir de sites duvidosos. Você simplesmente procura e instala o software que você precisa com o Gestor de Software.
Suporte

Se você é novo para o SolydXK ou sempre que você ficar bloqueado, há a comunidade SolydXK para ajudá-lo a resolver o seu problema. Você pode procurar o fórum para descobrir se o problema já foi resolvido ou iniciar um novo tópico para obter ajuda.
Forum: http://forums.solydxk.com
