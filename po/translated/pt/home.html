Aplicações domésticas

Esta é uma lista de aplicações recomendadas e adaptadas para utilizadores domésticos.

Quando você clica no botão instalar todos os aplicativos listados serão instalados no seu sistema. Você pode usar o gestor de software para instalá-los separadamente.
Gimp

GIMP é um acrônimo para GNU Image Manipulation Program. É um programa livremente distribuído para tarefas como o retoque de fotos, composição de imagem e imagem de autoria.

Tem muitas capacidades. Ele pode ser usado como um simples programa de pintura, um programa de retoque fotográfico com qualidade de especialistas, um sistema de processamento em carga on-line, um renderizador de imagem, produção em massa, um conversor de formato de imagem, etc.
Página inicial: http://www.gimp.org
Inkscape

Inkscape é um editor open-source de gráficos vetoriais semelhante ao Adobe Illustrator, Corel Draw, Freehand, ou Xara X. O que define a diferença do Inkscape é a sua utilização de Scalable Vector Graphics (SVG), um baseado em XML padrão W3C abertos, como o formato nativo.
Página inicial: https://inkscape.org
Shotwell / Digikam

Shotwell é um organizador de imagens projetado para fornecer uma gestão de fotos pessoais.

Shotwell pode importar fotos e vídeos de uma câmera digital diretamente. Shotwell agrupa automaticamente as fotos e vídeos por data, e suporta etiquetagem. Seus recursos de edição de imagens permitem aos utilizadores endireitar, cortar, eliminar olhos vermelhos e ajustar os níveis e equilíbrio de cores.

Para SolydxK Digikam está instalado. Ele tem a mesma funcionalidade que Shotwell

Página inicial do Shotwell: https://wiki.gnome.org/Apps/Shotwell

Página inicial do Digikamm: https://www.digikam.org
Clementine / Amarok

Clementine é um leitor de música moderno e organizador de biblioteca.

Clementine é um leitor de música plataforma. É inspirado pelo Amarok 1.4, com foco numa interface fácil de usar e rápido para procurar e tocar a sua música.

Para SolydxK o Amarok está instalado. Ele tem a mesma funcionalidade que o Clementine

Página inicial do Clementine: https://www.clementine-player.org

Página inicial do Amarok: https://amarok.kde.org
HandBrake

HandBrake é uma ferramenta para conversão de vídeo de quase qualquer formato para a seleção de codecs modernos, amplamente apoiadas.
Página inicial: https://handbrake.fr
SoundConverter

SoundConverter é o líder conversor de arquivos de áudio. Ele lê tudo o que o GStreamer pode ler (Ogg Vorbis, AAC, MP3, FLAC, WAV, AVI, MPEG, MOV, M4A, AC3, DTS, ALAC, MPC, Shorten, APE, SID, MOD, XM, S3M, etc ... ), e escreve para o Opus, Ogg Vorbis, FLAC, WAV, AAC, e arquivos MP3.
Página inicial: http://soundconverter.org
Pitivi / Kdenlive

Pitivi é um editor de vídeo gratuito com uma interface bonita e intuitiva, uma base de código limpo e uma comunidade fantástica.

Para SolydxK o Kdenlive está instalado. Ele tem a mesma funcionalidade que Pitivi

Página inicial Pitivi: http://www.pitivi.org

Página iniicial Kdenlive: https://kdenlive.org
Transmission / Ktorrent

Transmission é um cliente BitTorrent, que apresenta uma interface simples em cima de uma multi-plataforma de back-end.

Para SolydK o Ktorrent está instalado. Ele tem a mesma funcionalidade que o Transmission

Página inicial Transmission: https://www.transmissionbt.com

Página inicial Ktorrent: http://www.ktorrent.org
