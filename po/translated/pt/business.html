Aplicativos de negócios

Esta é uma lista de aplicativos recomendados adequados para as empresas.

Quando você clica no botão instalar todos os aplicativos listados serão instalados no seu sistema. Você pode usar o gestor de software para instalá-los separadamente.
Gnucash

GnuCash é um software de contabilidade financeira pessoal e de pequenas empresas.

Projetado para ser fácil de usar, mas poderoso e flexível, GnuCash permite que você acompanhe as contas bancárias, ações, renda e despesas. Tão rápido e intuitivo de usar como um registo de cheques, é baseado em princípios de contabilidade profissionais para assegurar livros equilibrados e relatórios precisos.
Pagina inicial: http://www.gnucash.org
Dolibarr

Dolibarr ERP e CRM é um software moderno e fácil de usar web para gerar os seus negócios (contatos, faturas, encomendas, produtos, stocks, agenda, emailings, etc ...).

Na primeira execução, a configuração final é iniciada. Você terá que preencher as seguintes informações:

  • Senha para o proprietário da base de dados Dolibarr: Dolibarr
  • Acesso de super-utilizador: root / your_root_password
  • Dolibarr administrador login : admin / admin

Substituir "your_root_password" com a sua senha de root.
Página inicial: http://www.dolibarr.org
Letodms

LetoDMS é um Sistema de Gestão de Documentos(DMS) de código aberto baseado na web, escrito em PHP com um backend de banco de dados. LetoDMS fornece metadados de documentos, controle de versão, segurança e facilidade de acesso aos seus documentos.

Login com: admin / admin

Você pode manter as bases de dados e Dolibarr Letodms com phpMyAdmin (Login: root / your_root_password)
Página inicial: http://www.letodms.com
ProjectLibre

Projectlibre é um sistema de software de gestão de projetos. Ele pretende ser um substituto de desktop completo para Microsoft Project.
Página inicial: http://www.projectlibre.org
