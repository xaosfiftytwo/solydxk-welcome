Aplicacions domèstiques

Aquesta és una llista d'aplicacions recomanades per a usuaris domèstics.

En clicar el botó d'instal·lació s'instal·laran al vostre sistema totes les aplicacions de la llista. Podeu usar el Gestor de Programari per instal·lar-les separadament.

Gimp

GIMP és un acrònim de GNU Image Manipulation Program (Programa de Manipulació d'Imatges GNU). És un programari lliure per tasques com el retoc de fotos, composició d'imatges i edició d'imatges de mapa de bits.

Té moltes aplicacions. Es pot usar com un simple programa de dibuix, programa de retoc fotogràfic, sistema de processat gràfic per lots, reproductor d'imatges en producció massiva, convertidor de formats, etc.
Pàgina Web: http://www.gimp.org
Inkscape

Inskape és un programa editor de gràfics vectorials de codi obert semblant a Adobe Illustrator, Corel Draw, Freehand o Xara X. El que fa diferent a Inkscape és l'ús de SVG (Scalable Vector Graphics), un estàndard obert W3C amb base XML com a format nadiu.
Pàgina Web: https://inkscape.org
Shotwell / Digikam

Shotwell és un organitzador d'imatges dissenyat per a la gestió personal de fotografies.

Shotwell pot importar directament fotos i vídeos d'una càmera digital. Shotwell agrupa automàticament les fotos i vídeos per data i suporta les etiquetes (tags). Les seves característiques d'edició d'imatges permeten als usuaris escalar, retallar, eliminar els ulls vermells i rectificar el color.

Per a SolydK s'instal·la Digikam. Té les mateixes funcionalitats que Shotwell.

Pàgina Web de Shotwell: https://wiki.gnome.org/Apps/Shotwell

Pàgina Web de Digikam: https://www.digikam.org
Clementine / Amarok

Clementine és un reproductor modern de música i organitzador de fonoteca.

Clementine és un reproductor de música multiplataforma. Està inspirat en Amarok 1.4, posant èmfasi en una interfície ràpida i fàcil per cercar i reproduir la vostra música.

Per SolydK s'instal·la Amarok. Té les mateixes funcionalitats que Clementine.

Pàgina Web de Clementine: https://www.clementine-player.org

Pàgina Web d'Amarok: https://amarok.kde.org
HandBrake

Handbrake és una eina per convertir vídeo de gairebé qualsevol format a una selecció de còdecs moderns amb suport ampli.
Pàgina Web: https://handbrake.fr
SoundConverter

SoundConverter és un convertidor de formats d'àudio capdavanter. Llegeix qualsevol cosa que GStreamer pugui llegir (Ogg Vorbis, AAC, MP3, FLAC, WAV, AVI, MPEG, MOV, M4A, AC3, DTS, ALAC, MPC, Shorten, APE, SID, MOD, XM, S3M, etc...) i ho enregistra en fitxers Opus, Ogg Vorbis, FLAC, WAV, AAC, i MP3.
Pàgina Web: http://soundconverter.org
Pitivi / Kdenlive

Pitivi és un editor de vídeo lliure amb una interfície intuïtiva i bonica, un codi de base net i una comunitat fantàstica.

Per SolydK s'instal·la Kdenlive. Té la mateixa funcionalitat que Pitivi.

Pàgina Web de Pitivi: http://www.pitivi.org

Pàgina Web de Kdenlive: https://kdenlive.org
Transmission / Ktorrent

Transmission és un client de BitTorrent amb una interfície senzilla sobre un potent motor multi-plataforma.

Per SolydK s'instal·la KTorrent. Té la mateixa funcionalitat que Transmission.

Pàgina Web de Transmission: https://www.transmissionbt.com

Pàgina Web de Ktorrent: http://www.ktorrent.org
