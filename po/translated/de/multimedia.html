Multimedia-Codecs

Diese folgenden Multimedia-Codecs sind aus rechtlichen oder aus Gründen der Sicherheit nicht enthalten.
Flash

HTML5 ist der Standardmediaplayer in Firefox.

Youtube und andere Websites bieten ihren Inhalt bereits im HTML5 Format an. Allerdings gibt es immer noch Webseiten, die allein auf Flash setzen.

Warnung:

Beachten Sie, dass Flash nicht Open Source ist. Einige Flash-Inhalte könnten Ihr System beschädigen.
CSS

Die meisten kommerziellen DVDs sind mit CSS (das Content Scramble System) verschlüsselt. um die Software welche für die Wiedergabe einer DVD verantwortlich ist, in ihrem Funktionsumfang einzuschränken.

In SolydXK sind die für das Abspielen dieser verschlüsselte DVDs notwendigen Pakete nicht vorinstalliert.

Durch Anklicken der Installationschaltfläche unten werden die folgenden Pakete auf Ihrem System installiert werden:

  • libdvdcss2
  • w32codecs or w64codecs

Rechtlicher Hinweis:

Konsultieen Sie Ihre lokalen Gesetze, um sicherzustellen, dass die Nutzung von libdvdcss2 rechtmäßig ist.
