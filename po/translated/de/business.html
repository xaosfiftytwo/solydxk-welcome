Geschäfts-Anwendungen

Das ist eine Liste von geeigneten Business Anwendungen.

Wenn Sie "Installieren" auswählen, werden alle aufgeführten Programme auf Ihrem System installiert. Einzeln können sie mit dem Software Manager installiert werden.
Gnucash

GnuCash ist eine Finanzbuchhaltungssoftware für Heimanwender und kleine Unternehmen.

Komplettlösung zur Verwaltung der Finanzen von Privatanwendern und Kleinbetrieben, basierend auf dem Prinzip der doppelten Buchführung. Homebanking wird ebenso unterstützt wie die Verwaltung von Kunden, Lieferanten und Rechnungen, Import von OFX und QIF, zahlreiche Berichte und Grafiken und eine benutzerdefinierbare Oberfläche.
Homepage: http://www.gnucash.org
Dolibarr

Dolibarr ERP & CRM ist eine moderne und leicht zu bedienende webbasierte Software zur Verwaltung von Unternehmen oder Vereinen (Kontakte, Mitglieder, Rechnungen, Aufträge, Produkte, Lagerbestand, E-Mails usw.).

Beim ersten Durchlauf wird die endgültige Konfiguration gestartet. Sie müssen die folgenden Informationen angeben:

  • Passwort für den Dolibarr Datenbankbenutzer: dolibarr
  • Superuser Zugang: root / Ihr Root Passwort
  • Dolibarr Administrator-Login : admin / admin

Ersetzen Sie Ihr "your_root_password" mit Ihrem Root-Passwort.
Homepage: http://www.dolibarr.org
Letodms

LetoDMS ist ein quelloffenes Web-basiertes Dokumentenmanagementsystem (DMS) mit einem Datenbank-Backend geschrieben, geschrieben in PHP. LetoDMS bietet Dokument-Metadaten, Versionskontrolle, Sicherheit und einfachen Zugriff auf Ihre Dokumente.

Login mit: admin / admin

Sie können die Dolibarr und Letodms Datenbanken mit phpMyAdmin verwalten (Benutzername: root / Root-Passwort)
Homepage: http://www.letodms.com
ProjectLibre

ProjectLibre ist ein Projekt-Management-System, das anstrebt, ein vollständiger Desktop-Ersatz für Microsoft Project zu sein.
Homepage: http://www.projectlibre.org
