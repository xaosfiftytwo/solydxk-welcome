Wenn Sie "Installieren" auswählen, werden alle aufgeführten Programme auf Ihrem System installiert. Einzeln können sie mit dem Software Manager installiert werden.
PlayOnLinux

SolydXK enthält eine Vielzahl von Programmen. Sollten Sie jedoch in den Paketquellen nicht das richtige finden, können Sie auch MS Windows-Software mit Hilfe von Wine und PlayOnLinux installieren. Eine große Anzahl von Software wird bereits unterstützt, aber dennoch funktioniert das nicht immer..

Sie können sich eine Anleitung zur Installation von MS Office 2010 mit PlayOnLinux auf unserer Webseite ansehen.

Durch die Nutzung dieser Software können Sie Ihr System anfällig für MS Windows Viren und Malware zu machen. Das Infektionsrisiko ist aber nicht höher als das unter einem nativen MS Windows.

Stellen Sie sicher, dass Sie Software nur aus vertrauenswürdigen Quellen installieren.
