Wenn Sie den Installationsschaltfläche anklicken, werden alle aufgeführten Programme auf Ihrem System installiert. Sie können den Software Manager benutzen, um diese gesondert zu installieren.
PlayOnLinux

SolydXK kommt mit einer Vielzahl von Softwareprogrammen. Sollten Sie jedoch die richtige Software in unserem Repository nicht finden können und Sie MS Windows-Software verwenden müssen, dann können Sie diese Programmen mit PlayOnLinux installieren. Nicht alle Software kann auf diese Weise ausgeführt werden, aber eine große Anzahl von Software wird bereits unterstützt.

Sie können sich eine Anleitung auf mit dem Namen Installation von MS Office 2010 mit PlayOnLinux auf unserer Webseite ansehen.

Durch die Nutzung dieser Software können Sie Ihr System anfällig für MS Windows Viren und Malware zu machen. Das Infektionsrisiko ist nicht höher als das unter einem nativen MS Windows.

Stellen Sie sicher, dass Sie Software nur von vertrauenswürdigen Quellen installieren.
