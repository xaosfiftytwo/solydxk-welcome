LibreOffice (Backports)

Wenn Sie den Installationsschaltfläche anklicken, werden alle aufgeführten Programme auf Ihrem System installiert. Sie können den Software Manager benutzen, um diese gesondert zu installieren.
LibreOffice

SolydXK kommt mit der Libreoffice-Suite vorinstalliert. Ihre Stabilität und Ausstattung ist für die meisten Menschen und vor allem für Unternehmen gut geeignet. Sollten Sie allerdingseinige Funktionen vermissen oder jeweils die aktuelleste Version vonLibreoffice verwenden, können Sie Libreoffice aus dem Backports-Repository installieren.

Backports sind extra kompilierte neue Pakete für die aktuelle stabile Umgebung. Dadurch können sie ohne neue Bibliotheken laufen.

Sie können zwar auch darüberhinaus die jeweils neueste Software aus dem Backports-Repository installieren, jeoch erhöht sich dadurch das Risiko von Inkompatiblitäten.
Homepage: https://www.libreoffice.org
