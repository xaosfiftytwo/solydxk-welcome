Systemanwendungen

Wenn Sie "Installieren" auswählen, werden alle aufgeführten Programme auf Ihrem System installiert. Einzeln können sie mit dem Software Manager installiert werden.
Gparted

GParted ist ein freier Partitioneditor zur grafischen Verwaltung Ihrer Festplattenpartitionen. Mit GParted können Sie Partitionen ohne Datenverlust kopieren und verschieben sowie ihre Größe ändern.
Homepage: http://gparted.sourceforge.net
VirtualBox

VirtualBox ist eine Virtualisierungsumgebung für x86-Hardware und ist auf Server-, Desktop- und Embedded-Einsatz ausgerichtet. Das Paket installiert sich auf einem vorhandenen Host-Betriebssystem als Anwendung, die dann den Start und die Ausführung zusätzlicher Gastbetriebssysteme ermöglicht, jedes mit seiner eigenen virtuellen Umgebung.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client ist ein schneller und zuverlässiger Cross-Plattform-FTP, FTPS und SFTP-Client mit vielen nützlichen Funktionen und einer intuitiven grafischen Benutzeroberfläche.
Homepage: https://filezilla-project.org
