SolydXK wordt geleverd met een breed scala aan software. Echter, als u de juiste software in onze repository niet kunt vinden en u moet toch MS Windows software gebruiken, kunt u deze installeren met behulp PlayOnLinux. Niet alle software kan op deze manier gebruikt worden, maar een grote hoeveelheid software wordt al ondersteund.

U kunt op onze site een tutorial bekijken over het installeren van MS Office 2010 met PlayOnLinux .

Door deze software te gebruiken, kunt u uw systeem kwetsbaar maken voor MS Windows virussen en malware. Het risico van infectie is niet hoger dan onder MS Windows zelf.

Zorg ervoor dat u alleen software van vertrouwde partijen installeert.
