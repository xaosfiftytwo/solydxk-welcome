Applicazioni business

Questa è una lista di applicazioni consigliate per l'uso business

Facendo clic su "installa", le applicazioni selezioante verranno installate nel sistema. È possibile usare il software manager per installarle sepratamente.
Gnucash

GnuCash è un software per la gestione economica e finanziaria personale e per piccole aziende.

Gnucash fornisce funzioni di contabilità adatte per la piccola impresa e per uso personale. Può tener traccia dei dati finanziari in diversi conti, registrando il saldo corrente e quello riconciliato; ha la gestione di clienti, venditori e dipendenti. Ha un'interfaccia grafica basata su X, gestisce la partita doppia, un'organizzazione gerarchica di conti, categorie di spesa, e può importare file Quicken QIF e OFX.
Homepage: http://www.gnucash.org
Dolibarr

Dolibarr ERP e CRM è un moderno sofware web semplice da usare e da mantenere per il business (contatti, chiamate, ordini, prodotti, stocks, agenda, email, ecc...).

Al primo avvio, la configurazione finale viene avviata. Sarà necessario inserire le seguenti informazioni:

  • Password per il database Dolibarr: dolibarr
  • Accesso superuser: root / la_password_di_root
  • Login per admin in Dolibarr: admin / admin

Sostituire "la_password_di_root" con la password di root.
Homepage: http://www.dolibarr.org
Letodms

LetoDMS è un sistema di gestione documentale basato su PHP e MySQL che permette di gestire meta-data, version control, sicurezza e accesso facile ai documenti.

Accedere con: admin / admin

È possibile gestire i database di Dolibarr e Letodms con phpMyAdmin (Login: root / la_password_di_root)
Homepage: http://www.letodms.com
ProjectLibre

ProjectLibre è un software per pianificare e gestire progetti aziendali. È compatibile con Microsoft Project, di cui vuole essere un sostituto.
Homepage: http://www.projectlibre.org
