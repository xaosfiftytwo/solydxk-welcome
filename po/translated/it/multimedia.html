Multimedia codecs

Questi multimedia codecs non sono inclusi di default per motivi legali o di sicurezza.
Flash

Il motore di default per la riproduzione di contenuti multimediali in Firefox è attraverso HTML5.

Youtube e altri siti web forniscono supporto alla visualizzazione con HTML5. Tuttavia alcuni siti web potrebbero ancora richiedere di usare Flash.

Attenzione:

Flash non è software Open Source. È possibile che alcuni contenuti Flash possano danneggiare il sistema.
CSS

La maggior parte dei DVD commerciali sono cifrati con il sistema di protezione CSS (Content Scramble System) che tenta di restringere il numero software in grado di riprodurre DVD.

SolydXK non fornisce in modo predefinito i pacchetti necessari per riprodurre tali DVD criptati.

Cliccando sul pulsante "installa", i seguenti pacchetti verranno installati:

  • libdvdcss2
  • w32codecs oppure w64codecs

Nota legale:

Controllare con la normativa locale ed assicurarsi che l'utilizzo di libdvdcss2 sia legale nell'area attuale.
