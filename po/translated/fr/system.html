Applications système

Si vous cliquez sur 'Installation', toutes les applications seront installées sur votre système. Vous pouvez les installer séparément en utilisant le gestionnaire de logiciels.
Gparted

GParted est un éditeur de partitions libre permettant de gérer graphiquement les partitions disques. Avec GParted vous pouvez redimensionner, copier et déplacer vos partitions sans pertes de données.
Page d'accueil: http://gparted.sourceforge.net
VirtualBox

VirtualBox est un logiciel de virtualisation généraliste pour matériel de type x86, à destination des serveurs, machines de bureau et usage embarqué. Le paquet VirtualBox s'installe sur un système d'exploitation hôte comme une application; cette application permet de charger et faire fonctionner des systèmes d'exploitation supplémentaires. Chacun est reconnu comme un sytème d'exploitation invité, avec son propre environnement virtuel.
Page d'accueil: https://www.virtualbox.org
FileZilla

FileZilla Client est un client FTP, FTPS et SFTP, multi-plateforme rapide et fiable. Il possède de nombreuses fonctionnalités intéressantes et une interface graphique intuitive.
Page d'accueil: https://filezilla-project.org
