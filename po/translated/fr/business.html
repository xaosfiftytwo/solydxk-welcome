Applications professionnelles

Voici une liste d'applications recommandées spécifiques aux entreprises.

Si vous cliquez sur 'Installation', toutes les applications seront installées sur votre système. Vous pouvez les installer séparément en utilisant le gestionnaire de logiciels.
Gnucash

GnuCash est un gestionnaire de finances personnelles.

Conçu pour être simple d'utilisation, puissant et flexible, GnuCash vous permet de suivre l'évolution de vos comptes bancaires, portefeuilles d'actions, revenus et dépenses. Aussi rapide et intuitif à utiliser qu'un registre de compte papier, il est basé sur des principes de comptabilité professionnelle pour assurer l'équilibre des comptes et des rapports fiables.
Page d'accueil: http://www.gnucash.org
Dolibarr

Dolibarr ERP & CRM est un logiciel moderne de gestion de votre activité professionnelle ou associative (contacts, factures, commandes, stocks, agenda, etc...).

Au premier démarrage, la configuration définitive doit être mise en place en entrant les informations suivantes:

  • Mot de passe du propriétaire de la base de données Dolibarr: dolibarr
  • Accès superutilisateur: root / votre_MDP_root
  • Login administrateur Dolibarr: admin / admin

Remplacez "votre_MDP_root" par votre mot de passe root.
Page d'accueil: http://www.dolibarr.org
Letodms

LetoDMS est un logiciel libre de gestion documentaire (DMS) en ligne écrit en PHP et comportant une base de données. LetoDMS fournit les métadonnées, le contrôle des versions, la sécurité et un accès facilité à vos documents.

Connexion avec: admin /admin

Vous pouvez gérer les bases de données de Dolibarr et Letodms avec phpMyAdmin (Login: root / votre_mot_de_passe)
Page d'accueil: http://www.letodms.com
ProjectLibre

ProjectLibre est un outil de planification de projet. Il vise à se substituer complètement à Microsoft Project.
Page d'accueil: http://www.projectlibre.org
